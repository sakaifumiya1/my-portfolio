# Sports Trainer Portfolio

スポーツトレーナー向けのポートフォリオサイトです。

## 概要

このポートフォリオサイトは、スポーツトレーナーとしての実績、スキル、サービスを紹介するためのWebサイトです。モダンでレスポンシブなデザインで、アスリートやクライアントにプロフェッショナルな印象を与えます。

## 特徴

- 🎨 モダンで洗練されたUIデザイン
- 📱 完全レスポンシブ対応
- ✨ インタラクティブなアニメーション効果
- 🎯 パフォーマンス最適化
- 🌐 日本語対応

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)

## セクション

- **ホーム**: ヒーローセクションと自己紹介
- **自己紹介**: プロフィールと統計情報
- **スキル**: 専門スキルの紹介
- **実績・プログラム**: 過去のプロジェクトと実績
- **お問い合わせ**: 連絡先情報とコンタクトフォーム

## セットアップ

1. リポジトリをクローン:
```bash
git clone https://github.com/your-username/my-portfolio.git
cd my-portfolio
```

2. ブラウザで`index.html`を開く

## カスタマイズ

- `index.html`: コンテンツの編集
- `styles.css`: スタイルのカスタマイズ
- `script.js`: インタラクティブ機能の追加
