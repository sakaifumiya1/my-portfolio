# Sports Trainer Portfolio

スポーツトレーナー向けのポートフォリオサイトです。

## 特徴

- モダンでレスポンシブなデザイン
- スムーズなアニメーションとインタラクティブな要素
- パーティクル背景エフェクト
- カスタムカーソル
- タイピングアニメーション

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/YOUR_USERNAME/my-portfolio.git
```

2. プロジェクトディレクトリに移動
```bash
cd my-portfolio
```

3. ブラウザで `index.html` を開く
