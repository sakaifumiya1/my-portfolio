# Sports Trainer Portfolio

スポーツトレーナー向けのポートフォリオサイトです。

## 概要

このプロジェクトは、モダンでレスポンシブなポートフォリオサイトです。

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)

## セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/YOUR_USERNAME/my-portfolio.git
```

2. プロジェクトディレクトリに移動
```bash
cd my-portfolio
```

3. ブラウザで `index.html` を開く
